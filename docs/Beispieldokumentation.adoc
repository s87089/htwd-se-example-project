== Einführung
Dieses Dokument dient im Praktikum als Spielwiese für das Kennenlernen und praktische Üben der Syntax von AsciiDoc.

== Kennenlernen der Syntax

Über die Aufgaben wird das Dokument Stück für Stück aufgebaut und angepasst.

=== Listen

.Name: unsortierte Liste
.Beispiel: unsortierte Liste 

* Level 1 list item
** Level 2 list item
*** Level 3 list item
**** Level 4 list item
***** Level 5 list item
****** etc.
* Level 1 list item

.Name: sortierte Liste
.Beispiel: sortierte Liste
    	
. Level 1 list item
.. Level 2 list item
... Level 3 list item
.... Level 4 list item
..... Level 5 list item
. Level 1 list item

=== Tabellen und Textformatierung
.Textformatierung
|===
|Formatierung| Sytax| Beschreibung

| *bold* | +*bold*+ | fetter Text
| _italic_ | +_italic_+ | kursiver Text
| `monospace` | +`monospace`+ | Text mit fester Laufweite
| *_bolditalic_* |  +*_bolditalic_*+ | Text mit kombinierter Formatierung
| **bold**__italic__ | +**bold**__italic__+| Text mit Formatierung innerhalb eines Wortes
|===

.Beispiel: Software Engeneering

**S**oftware **E**ng__i__n__ee__``ring``
